Features

Everything you need to split audio

🤖

Powered by Demucs AI

Uses the state-of-the-art Demucs model by Meta Research — the same technology used by audio professionals worldwide.

📦

Fully Self-Contained

Ships with its own embedded Python runtime and all dependencies. Download, run, done. No installation, no Python required.

⚡

GPU Acceleration

Supports CUDA-enabled GPUs for significantly faster processing. Falls back to CPU automatically if no GPU is detected.

🎵

Multiple Formats

Works with MP3, WAV, FLAC, OGG and more. Outputs high-quality stems you can drop straight into your DAW.

🖥️

Clean Native UI

Simple drag-and-drop interface built with PySide6. No browser, no Electron — a real native Windows application.

🆓

Completely Free

No subscriptions, no limits, no watermarks. Process as many tracks as you want, locally on your own machine.

How it works

Three steps, that's it

1

Download

Grab the single .exe file — no installer needed.

2

Load your track

Drag and drop any audio file or browse to select it from your drive.

3

Get your stems

Hit split and wait. Each stem is saved as a separate file, ready to use.

Requirements

What you need

Windows 10 or 11 64-bit only
~4 GB free disk space For the app and AI models
100% offline AI models are bundled — no internet required
GPU optional CUDA 11.8+ for faster processing
